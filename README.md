    weak_ptr을 사용해 이벤트 실행 대상의 생존 여부를 확인하고, 파괴된 객체에 접근하는 문제를 방지했습니다.

    드래그 앤 드롭으로 참조를 연결하는 장면

    인스펙터에서는 타입 기반 드래그 앤 드롭, 검색, 선택, 해제를 지원해 참조 연결 작업을 데이터 편집으로 처리했습니다.
