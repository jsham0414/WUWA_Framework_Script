    weak_ptr을 사용해 이벤트 실행 대상의 생존 여부를 확인하고, 파괴된 객체에 접근하는 문제를 방지했습니다.

#### 래퍼 시스템

- 시간  
    3:05 ~ 3:40

- 화면

    드래그 앤 드롭으로 참조를 연결하는 장면

    런타임에서는 포인터로 접근하고, 저장 시에는 GUID 또는 Path로 직렬화해 씬/프리팹 로드 후 참조를 복원합니다.

    WObject는 토큰으로 대상 객체의 파괴 여부를 확인해 댕글링 참조를 방지하고, WAsset은 지연 로딩과 참조 카운팅으로 에셋 생명주기를 관리합니다.

    인스펙터에서는 타입 기반 드래그 앤 드롭, 검색, 선택, 해제를 지원해 참조 연결 작업을 데이터 편집으로 처리했습니다.
